Berdasarkan dokumen pedoman tersebut, berikut adalah ringkasan khusus mengenai **Pedoman Penulisan Kode Program**:

### 1\. Penempatan dalam Dokumen

- Kode program (Listing Program) diletakkan pada bagian **Lampiran**.
- Tujuannya agar data tersebut tidak mengganggu kesinambungan pembacaan isi utama tugas akhir.

### 2\. Format Font dan Ketikan

- **Jenis Huruf (Font)**: Secara umum menggunakan font **Courier New**.
- **Ukuran Huruf**: Menggunakan ukuran **8 pt** jika diletakkan di bagian lampiran.
- **Jarak Spasi**: Menggunakan **spasi tunggal** (_line spacing = single_).
- **Kerapian**: Harus diketik dengan rapi dan menggunakan format rata kiri kanan (_justify_).
- **Format**: Dikurung menggunakan tabel 1:1

### 3\. Penomoran Halaman

- Halaman yang berisi kode program di bagian lampiran harus diberi nomor halaman dengan format **L-x**.
- Nomor halaman tersebut diletakkan di posisi **tengah bawah**.

### 4\. Referensi dalam Daftar Lampiran

- Keberadaan kode program harus dicantumkan dalam **Daftar Lampiran** dengan keterangan "Listing Program".
- Setiap kode program yang disertakan harus merujuk pada nomor halaman lampiran yang sesuai (misalnya: L-1).
