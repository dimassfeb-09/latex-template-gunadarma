\section{PENUTUP}
\onehalfspacing

\subsection{Kesimpulan}
\justifying

Berdasarkan hasil penelitian dan pembahasan, dapat ditarik kesimpulan sebagai berikut:

\begin{enumerate}
    \item Kesimpulan Pertama
    \item Kesimpulan Kedua
    \item Kesimpulan Ketiga
\end{enumerate}

\subsection{Saran}
\justifying

Adapun saran untuk pengembangan selanjutnya adalah:

\begin{enumerate}
    \item Saran Pertama
    \item Saran Kedua
\end{enumerate}
